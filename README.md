# PolyTrade

A terminal trading platform for Polymarkets' crypto "next candle" yes/no events.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Terminal trading interface
- Strategy builder and management
- Analytics dashboard
- Trading history
